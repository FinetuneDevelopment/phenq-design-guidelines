Bleed grid

Bleed grids are a hybrid of full-bleed layouts and the standard grid. They only allow for two column layouts, but are configurable so that the full-bleed image can sit to the right or left.

Note that bleed grid mark-up must be placed outside of any container elements.





Configuring

The background colour of the bleed grid is selected using the classes which can be found on the colour guidelines page.

The height of the bleed grid is defined by the content. The image will stretch or shrink to fit within whatever space is left.

Images could be added via class or as inline styles. The alignment of each image may require unique configurations.



Bleed bisect

Bleed bisect must sit outside of .container and for the best visual effect, requires the following markup structure:

  • .bleed-bisect
    • .bleed-bisect__child-right (for example)
    • .container
      • .cta-box .radius-big

Note that both the .bleed-bisect__child-right and .cta-box should have the same background colour class applied to them.

Off-grid right

Elements within container can have either their left or right margins extended to the edge of the page.

Note that the JavaScript assumes the distance to the left side of the page is the same as the distance to the right side of the page.

The element should have an data-offgrid="right" attribute.
