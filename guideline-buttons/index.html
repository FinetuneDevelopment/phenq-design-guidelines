Secondary buttons

Default size

Note that in order for the gradient border to work, the element cannot be empty. This means that if it is to be used on a submit button for a form, that button must be a button tag, rather than a input which has a type of submit. If the btn-secondary class is used on an empty element, it will fall back to a bright blue border.

The gradient border is added to a :before pseudo-element, which is assigned a z-index of -1. This means that if a secondary button is used inside an element which has a background added to it, there is a chance that the gradient background of the button will fall behind the background colour (or image) in the z-order. In this context, the border of the button will fall back to a solid colour.
